C#

C# est l’un des langages de programmation les plus populaires. Il peut être utilisé pour une variété de choses, y compris les applications mobiles, le développement de jeux et les logiciels d’entreprise. Connaître C# vous ouvre de nombreuses portes en tant que développeur.

Le langage de programmation C# a été conçu par Anders Hejlsberg de Microsoft en 2000 et a ensuite été approuvé en tant que norme internationale par l’Ecma (ECMA-334) en 2002 et l’ISO/IEC (ISO/IEC 23270 et 20619) en 2003. Microsoft a introduit C# ainsi que .NET Framework et Visual Studio, qui étaient tous deux à code source fermé. À l’époque, Microsoft n’avait pas de produits open source. Quatre ans plus tard, en 2004, un projet libre et open-source appelé Mono a commencé, fournissant un compilateur multiplateforme et un environnement d’exécution pour le langage de programmation C#. Une décennie plus tard, Microsoft a publié Visual Studio Code (éditeur de code), Roslyn (compilateur) et la plate-forme unifiée .NET (infrastructure logicielle), qui prennent tous en charge C# et sont gratuits, open source et multiplateformes. Mono a également rejoint Microsoft mais n’a pas été fusionné avec .NET.

À partir de novembre 2023, la version stable la plus récente du langage est C# 12.0, qui a été publiée en 2023 dans .NET 8.0.
Vous pouvez apprendre le C# avec d'autres languages comme codecademy.
